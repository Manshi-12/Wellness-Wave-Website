INSTALLATION:
==============================================================================
1. Clone the repository:
   $ git clone https://github.com/Manshi-12/Wellness-Wave-Website
   
2. Navigate into the project folder:
   $ cd wellness-wave

3. Open the `index.html` in your browser:
   $ open index.html
